Javascript Project - 2

Condition With Looping Programs

  1. Basic Level

    • Print all even numbers from 1 to 50 using a for loop.
    • Print numbers from 1 to 100, but skip multiples of 3 using continue.
    • Find the sum of all numbers from 1 to N (take N as input) using while.
    • Check and print whether each number from 1–20 is prime or not.
    • Print a multiplication table for a given number using a loop.
    • Count how many vowels are in a string using a for loop and if.
    • Reverse a number (e.g., 1234 → 4321) using while and modulus.
    • Find the factorial of a number using a loop.
    • Print the Fibonacci series up to N terms.
    • Check if a number is an Armstrong number (e.g., 153 = 1³+5³+3³).

  2. Real-World Style

    • Attendance checker: Loop through an array of daily attendance (P/A), count total presents, and print "Eligible for exam" if presents ≥ 75%.
    • Grade calculator: Loop through an array of marks, use if-else to assign grades (A/B/C/F) for each.
    • ATM cash dispenser: Given an amount, use a loop with if conditions to calculate the minimum number of 2000/500/100 notes.
    • Shopping cart discount: Loop through an array of item prices, apply discount slabs (10% if >1000, 5% if >500) using if-else.
    • Login attempt simulator: Use a while loop to allow max 3 password attempts, break on success.
    • Electricity bill generator: Loop through units consumed and apply slab-based rates (0–100 units @₹5, 101–200 @ ₹7, etc.).
    • Student pass/fail report: Loop through an array of student objects {name, marks}, print PASS/FAIL for each using if.

  3. Slightly Advanced

    • Print a right-angled triangle pattern of stars using nested loops.
    • Find the largest and smallest number in an array using a loop and if comparisons.
    • FizzBuzz: Print numbers 1–100, but print "Fizz" for multiples of 3, "Buzz" for multiples of 5, and "FizzBuzz" for both.
